# Vagrant setup for WordPress

## Steps

1. Create new project with this VagrantFile as a starting point.

		```sh
		$ git clone https://github.com/stedman/vagrant-wordpress <project_name>
		$ cd <project_name>
		```
2. Modify VagrantFile to meet your needs.
3. Start up Vagrant.

		```sh
		$ vagrant up
		```
